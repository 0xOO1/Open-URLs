# Open-URLs
- ⛅ Open URLs 谷歌浏览器插件 高效渗透工具 实现一键打开URL 解放双手
- ⛅ 可选择一次性打开xx个网页 缓解浏览器压力 默认全部打开 
- 📧 意见反馈：0xShe@t00ls.net

### 使用方法：
- 🍺 谷歌浏览器打开开发者模式 - 加载已解压的扩展程序 
- 🍺 文本框输入URL 一行一个 点击Go 即可一键全部访问
- 🍺 从此解放双手
